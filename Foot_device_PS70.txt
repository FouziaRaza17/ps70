Insoles with sensors
-Carbon-embedded piezoresistive material
between two laters of electrodes
-15 elements for horizontal; 5 elements for 
vertical electrodes --> 75 sensing nodes!
-Electrodes wired to microcontroller 
(TEENSY 3.1)
-Microcontroller set to run at 1000Hz == ~13 Hz 
per sensing node per insole 
-HC-05 Bluetooth modules used
